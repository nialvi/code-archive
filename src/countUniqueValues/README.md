# Count unique values

Needed to count unique values in a sorted array. Time complexity O(n). Space complexity O(n).

```javascript
countUniqueValues([1, 1, 1, 1, 1, 1, 2]); // 2
countUniqueValues([]); // 0
countUniqueValues([1, 1, 2, 2, 5, 6, 7, 7, 11]); // 6
countUniqueValues([-1, -1, -2, -2, -5, 6]); // 4
```
